Pessoal post rápido, para **renomear uma branch no git** basta executar.

{% highlight bash %}
git branch -m antigo_nome novo_nome
{% endhighlight %}

Onde `antigo_nome` é o nome atual da **branch** e o `novo_nome` é o nome que você quer que seja.

Rápido e prático!
**Git é amor!**
